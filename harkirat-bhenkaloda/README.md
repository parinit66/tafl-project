# Chomsky Hierarchy Visualization Tool

This project is a web-based visualization tool to help students understand the Chomsky hierarchy of languages. It features an interactive diagram, explanations, and examples for each class: Regular, Context-Free, Context-Sensitive, and Recursively Enumerable.

## Features
- Interactive diagram of the Chomsky hierarchy
- Explanations and examples for each language class
- Modern, responsive UI (Tailwind CSS)
- No login or setup required
- Ready for deployment on Vercel

## Getting Started

1. **Install dependencies:**
	```bash
	npm install
	```
2. **Run the development server:**
	```bash
	npm run dev
	```
3. **Open [http://localhost:3000](http://localhost:3000) to view the app.**

## Deployment
Deploy directly to Vercel for best results.

---

This project was bootstrapped with Next.js and Tailwind CSS.
